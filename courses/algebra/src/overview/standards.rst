Standards

:date: 2012-11-23
:summary: List of content standards for Algebra
:tags: standards
:category: overview



BI (Big Ideas):
   
   1. Make sense of problems and persevere in solving them. ( BI.1_ )
   2. Reason abstractly and quantitatively.  ( BI.2_ )
   3. Construct viable arguments and critique the reasoning of others. ( BI.3_ )  
   4. Model with mathematics.  ( BI.4_ )
   5. Use appropriate tools strategically. (BI.5_ )  
   6. Attend to precision.  ( BI.6_ )
   7. Look for and make use of structure. ( BI.7_ )  
   8. Look for and express regularity in repeated reasoning. ( BI.8_ )


AC (Algebra Content):

   1. Manipulating algebraic expressions ( AC.1_ )
   2. Solving linear equations( AC.2_ )
   3. Properties of real numbers( AC.3_ )
   4. Solving linear inequalities( AC.4_ )
   5. Absolute value( AC.5_ )
   6. Graphing( AC.6_ )
   7. Systems of equations( AC.7_ )
   8. Graphing systems( AC.8_ )
   9. Applications of systems of equations( AC.9_ )
   10. Determinants( AC.10_ )
   11. Exponents( AC.11_ )
   12. Factoring( AC.12_ )
   13. Applications of factoring( AC.13_ )
   14. Rational expressions( AC.14_ )
   15. Rational exponents( AC.15_ )
   16. Quadratic equations( AC.16_ )
   17. Applications of quadratics ( AC.17_ )
